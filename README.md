# NirogBot 🏥
An AI-powered multilingual public health chatbot built for Smart India Hackathon 2025.

- **Problem Statement ID:** 25049
- **Organization:** Government of Odisha, Electronics & IT Department
- **Theme:** MedTech / BioTech / HealthTech

---

## 🌟 Features
- 🌐 Multilingual support (English, Hindi, Odia, Tamil)
- 🤒 Symptom detection from local disease database (16 diseases)
- 🤖 AI-powered responses via Groq API (LLaMA 3.1)
- 🚨 Real-time cross-device outbreak alert system
- 📊 Analytics dashboard with disease frequency chart
- 💾 Chat history with localStorage
- ✍️ Typewriter effect for bot responses
- 🚩 Chat reporting system

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **AI:** Groq API (llama-3.1-8b-instant / llama-3.3-70b-versatile)
- **Database:** JSON + localStorage
- **Hosting:** Vercel (Frontend) + Render (Backend)
